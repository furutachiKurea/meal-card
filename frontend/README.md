# 前端

React + Vite + Ant Design

## 环境要求

- Node.js 18+
- pnpm 8+

## 安装依赖

```bash
pnpm install
```

## 启动开发服务器（监听 :5173）

```bash
pnpm dev
```

## 构建生产包

```bash
pnpm build
```

## 项目结构

```
frontend/
├── src/
│   ├── main.jsx        # 入口
│   ├── App.jsx         # 路由配置
│   ├── api.js          # API 封装
│   └── pages/          # 页面组件（7 个）
└── index.html
```
