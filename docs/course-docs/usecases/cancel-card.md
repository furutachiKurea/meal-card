# 注销

## 当前行为

操作员输入证件号，系统查找该证件号名下当前有效卡并展示卡信息和预计退款，操作员二次确认后执行注销，退还押金和余额。

## 前置条件

- 证件号名下存在 active 或 lost 状态的卡

## 成功路径

1. 操作员输入 12 位证件号
2. 系统按证件号查找当前有效卡（active 或 lost），展示卡号、持卡人姓名、余额、押金、预计退款合计
3. 操作员二次确认
4. 系统将卡状态改为 cancelled，余额清零
5. 返回退款明细：押金 + 原余额 = 应退总额

## 异常 / 边界情况

- 证件号无有效卡（只有 cancelled 或无记录）：返回"无有效卡"，流程中止
- 对已 cancelled 的卡再次注销：返回错误

## 规则 / 限制

- 操作入口为证件号（学号/工号），不直接输入卡号
- active 和 lost 状态的卡均可注销
- 注销后卡号永久停用，不可重新发卡（每次发卡生成新卡号）
- 退款金额 = 押金（固定 2000 分）+ 注销时的余额
