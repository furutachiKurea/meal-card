BroadcastChannel 核心代码

CODE 
// 顾客屏：刷卡后广播
const ch = new BroadcastChannel('window-1')
const res = await getCard(cardNo)
ch.postMessage({
  type: 'card_read',
  cardNo: res.cardNo,
  holderName: res.cardHolder.name,
  balance: res.balance
})

// 操作端：监听
ch.onmessage = (e) => {
  if (e.data.type === 'card_read') {
    setCardInfo(e.data)
  }
}
中文解释

new BroadcastChannel('window-1') — 打开对讲机，拨到频道 window-1

await getCard(cardNo) — 顾客屏先向后端验证卡片

ch.postMessage({...}) — 验证通过，把信息广播出去

ch.onmessage — 操作端监听，收到后更新界面
