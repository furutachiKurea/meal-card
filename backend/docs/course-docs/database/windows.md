# windows（窗口机）

## 作用
- 记录食堂各窗口机的信息，用于消费记录归属和按窗口统计

## 设计原因
- 需求中窗口机是独立实体，消费记录需要关联到具体窗口以支持"各窗口收入"统计
- 只保留必要字段，不做额外扩展

## 字段
- ID:
  - 含义: 窗口主键
  - 类型: uint
  - 是否必填: 自动生成
  - 默认值: 自增
- Name:
  - 含义: 窗口名称（如"一号窗口"）
  - 类型: string
  - 是否必填: 是
  - 默认值: 无

## 主键 / 索引 / 约束
- 主键: ID
- NOT NULL: Name

## 表关系
- 被 transactions 通过 window_id 引用（一对多）
