# deposit_records（存款记录）

## 作用
- 记录每次充值操作，用于存款明细查询和统计

## 设计原因
- 每次充值产生一条不可变记录，余额变动体现在 cards.Balance 上
- 用于统计"各持卡人存款明细"、"本日/月存款金额"等报表

## 字段
- ID:
  - 含义: 记录主键
  - 类型: uint
  - 是否必填: 自动生成
  - 默认值: 自增
- CardID:
  - 含义: 充值的饭卡
  - 类型: uint
  - 是否必填: 是
  - 默认值: 无
- Amount:
  - 含义: 充值金额
  - 类型: int64
  - 是否必填: 是
  - 默认值: 无
  - 备注: 单位：分，必须为正数（业务层校验）
- CreatedAt:
  - 含义: 充值时间
  - 类型: time.Time
  - 是否必填: GORM 自动填充
  - 默认值: 当前时间

## 主键 / 索引 / 约束
- 主键: ID
- 普通索引: CardID（按卡查询存款记录）
- NOT NULL: CardID, Amount
- 外键: CardID → cards.ID

## 表关系
- 属于 cards（多对一，通过 CardID）
