# 食堂饭卡管理系统

课设项目，前后端分离架构，实现发卡、存款、就餐消费、挂失、注销、汇总统计 6 项功能。

## 启动

```bash
# 后端（监听 :8080）
make dev-backend

# 前端（监听 :5173）
make dev-frontend

# 运行后端单测
make test
```

分两个终端分别运行 `dev-backend` 和 `dev-frontend`。
